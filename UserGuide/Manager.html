Manager - User Guides

Basically, a manger is considering as a cashier. So managers have all functions which cashiers have. Therefore, users can check the "cashier" page to know more about these functions. Besides that, managers also have some specific features which only managers have the permission to to that such as: create and manage the menu of the resturant, import and export order...

General View

In this view , managers can see menu with categories , menu items on the left, tables map on the right hand side and alll button needed for managers features to interact with menu, report, search, import/export CSV file , tables.
