Start Up - User Guides

All components and steps that's needed to run EzRestaurant java - desktop program will be listed and instructed here.

Step 1: Install JRE

JRE stands for Java Runtime Environment which allows Java written program can run on any flatform.

The EZRestaurant program is written in Java desktop so that the machine which is supposed to run the program must install JRE

User can download JRE via this link: JRE

Step 2: Install and run program

The program is provided to the customer by an installer. The user will use that file to install the program and there is a file which has .jar extension. After finish install the JRE to the computer, the machine should be able to execute the file to run the program.
There are 2 ways to execute the file:
- Double click the file.
- Run by command: java -jar filename.jar (filename is the name of the jar file).
